head标签内包含的标签

  • title标签：显示在浏览器的标题栏中（告诉用户和搜索引擎这个网页的主要内容）可以思考在SEO中的作用是什么？
  • meta标签
  • link标签

body内包含标签

- <span> <!--没有语义，作用是为了设置单独的样式用。(在style标签里面加入样式)-->
- <q></q> <!--语义为引用别人的话（简短文本），默认样式自带双引号-->
- <blockquote> <!--引用长文本，默认样式为缩进样式，无需输引号-->
- <br/> <!--换行，空标签-->
- &nbsp
- <hr/> <!--语义为添加水平横线-->
- <code> <!--单行代码-->
- <pre> <!--常用于代码段，但主要作用是预格式化的文本，通常保留空格和换行符-->
- ul-li <!--列表无前后顺序-->
- ol-li <!--有顺序的列表-->
- <div id=“…”>…</div> <!--id属性必须是唯一的-->

表格的四个元素：table、tbody、tr、th、td tbody标签使得表格全部下载完才能显示出来；tr是行；th是表头，默认样式是加粗居中；td是单元格

- <table summary=“….">
- <caption>
- <a href=“目标网址”  title=“鼠标滑过显示文本” target=“_blank">链接文本</a>
- <img src=“….” alt=“….” title=“…"/>

表单标签：

- <form method=“传送方式” action=“服务器文件”>...</form> <!--传送方式有get/post，一般使用post，较为安全；action是浏览者输入的数据被传送到的地方-->
    - <form>
        <input type=“text/password” name=“..." value=“…"/>
      </form>

name是为文本框命名，以备后台ASP/PHP调用；value为文本框设置默认值，一般起到提示作用

- <textarea cols=“…” rows=“…” >…..</textarea>

文本框和文本域的区别：文本框是input标签下的功能，通过type来控制，而文本框只是input的一个功能，比如还有input type=”radio”等，而文本域本身就是一个标签，是

- <input type=“radio/checkbox” value=“…” name=“…” checked=“checked”/>

注意单选框情况下，name值一定要一致才能实现单选

- 下拉列表框：
    <select>
    <option value=‘提交值’ selected=“selected">选项</option>
    </select>



- 下拉框多选
    <select multiple=“multiple">
    <option value=‘提交值’ >选项</option>
    </select>



- label标签：<label for=“控件id名称">
  • 注意：标签的 for 属性中的值应当与相关控件的 id 属性值一定要相同。 label标签的作用：没有任何特殊效果，为鼠标用户改进了可用性。当用户单击选中该label标签时，浏览器就会自动将焦点转到和标签相关的表单控件上（就自动选中和该label标签相关连的表单控件上) 注释语法：
    • 注重语义化，更容易被搜索引擎理解，更容易使得读者阅读*
