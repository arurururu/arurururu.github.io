CSS样式分类说明

定义：CSS样式由选择符和声明组成，而声明又由属性和值组成。
p{ color:blue }
多条声明之间用分号分隔

  • 内联式CSS样表：把CSS代码直接写在现有的HTML标签中，注意是开始标签，多条声明用分号隔开
  • 嵌入式CSS样表：把CSS样式代码写在以下标签之间，style标签在head标签之间
  • 外部式CSS样表：把CSS代码写在一个单独的外部文件中，以“.css”为扩展名，在head标签内使用link标签将CSS样式文件链接到HTML文件内

注意：

1.css样式文件名称以有意义的英文字母命名，如 main.css。

2.rel=”stylesheet” type=”text/css” 是固定写法不可修改。

3.link标签位置一般写在head标签之内。

三种样式优先级

三种样式的优先级：内联式>嵌入式>外部式
前提是相同权值下，一般情况下是这样，个人觉得应该顺从代码从上往下读的原则

关于选择器

每一条CSS样式声明由两部分组成：

选择器{
样式;
}
说明：选择器指明了样式的作用对象

选择器的分类以及区别

标签选择器

类选择器

.类选器名称{css样式代码;}

ID选择器

与类选择器类似
使用方法：

#ID名称{css样式代码;}

注意：

  • 为标签设置id=”ID名称”，而不是class=”类名称”。
  • ID选择符的前面是井号（#）号，而不是英文圆点（.）

类选择器与ID选择器区别：
相同点：可以用于任何元素
不同点：

  • ID选择器只能在文档中使用一次，而且仅一次。而类选择器可以使用很多次。
  • 可以使用类选择器词列表方法为一个元素同时设置多个样式，但不可以使用ID选择器（不能使用ID词列表）。

子选择器

即大于符号（>），用于选择指定标签元素的第一代子元素。
举例如下：
.food>li{border:1px solid red;}

包含（后代）选择器：即加入空格，用于选择指定标签元素下的后辈元素
.first span{color:red;}

总结：>作用于元素的第一代后代，空格作用于元素的所有后代。

通用选择器

说明：使用一个（*）号指定，匹配HTML中所有标签元素

* {color:red;}

伪类选择符

说明：允许给HTML不存在的标签（标签的某种状态）设置样式
常用鼠标滑过的状态来设置（兼容性最好）
a:hover{color:red;}

分组选择符

说明：一个逗号

h1,span{color:red;}

CSS样式的继承性

某些样式是具有继承性的

不可继承的属性如下：

display、margin、border、padding、background、height、min-height、max-height、width、min-width、max-width、overflow、position、left、right、top、 bottom、z-index、float、clear、table-layout、vertical-align、page-break-after、 page-bread-before和unicode-bidi。

CSS样式的特殊性

标签的权值为1，类选择符的权值为10，ID选择符的权值最高为100.

注意：还有一个权值比较特殊—继承也有权值但很低，有的文献提出它只有0.1，所以可以理解为继承的权值最低。

CSS样式的层叠性

说明：后面的样式会覆盖前面的样式。

CSS样式的重要性

说明：语法为
!important

样式优先级

浏览器默认的样式 < 网页制作者样式 < 用户自己设置的样式，但记住!important优先级样式是个例外，权值高于用户自己设置的样式。
